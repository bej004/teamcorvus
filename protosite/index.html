HTML5 is now a recommendation

16 Sep 2014

You might not realize this, but HTML5 has been a RFC (or "Request For Comment") for the last 5 years, which means that it wasn't an official specification. The recent move to Recommendation status means that HTML5 is now an official specification.

What does this mean?

Basically it means that you can now use it in any application and expect that the rules set out in the spec can (and should be) followed by developers and manufacturers.

Further reading

You can read more about the move to recommendation status or the announcement from the W3C.

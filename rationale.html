Design Decisions

During an initial group consultation, we explored the existing BCIT site, analyzed the requirements of the redesigned site, and decided on the following design principles.

  • Users want easy and direct access to lecture notes
  • A drop-down menu provides access to lecture notes in one click
  • Many users will access site from a mobile device, therefore site must be responsive
  • Users will want to print lecture notes, therefore a print stylesheet is required
  • The Accordian jQuery plugin will allow users to display and hide sections in order to aid learning
  • Contact details for the lecturer should be readily available on each page
  • HTML and CSS is to comply with web development standards
